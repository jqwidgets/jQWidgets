Description

This is an example of the drill through functionality of the Angular Pivot Grid . This feature allows you to find the records in the data source which were used to calculate the value of a particular cell in the pivot grid. To drill through a specific grid cell double click on it and you will see a window displaying the source records. In addition to the pivot grid component, this example uses the window and datatable components. The example also demonstrates how to handle open and close events of the window component in Angular and how to enable or disable the selection functionality of the pivot grid component.

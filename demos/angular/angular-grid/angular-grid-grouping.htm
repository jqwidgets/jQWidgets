This example demonstrates basic grouping in Angular 4 Grid and shows collapsible data groups that can be customized via the 'Group By' header menu option or via drag and drop of grid column headers. You have the options to expand or collapse the groups. The grouping can also work when the Pager feature is enabled. In addition you have the option to show aggregates. The aggregates are shown next to the groups or at the bottom of the grid.

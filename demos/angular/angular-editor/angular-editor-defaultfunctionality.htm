This is an example of Angular 4 Editor. The Angular editor is a ready for use HTML editor which can simplify web content creation or be a replacement of your HTML Text Areas. It supports text alignment � centering, left and right and indentations. You can also insert ordered lists. Text formatting will enable you to have bold, underlined and italic texts. You can format blocks with different headers. You can set the text and the background colors and change the font. In addition the Angular Editor supports Hyperlink and image dialogs. The toolbar position of the editor can be horizontal or vertical. You can have right to left layout and localization. The data of the editor can be printed using the standard browser printing functionality.

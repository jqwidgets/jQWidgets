jqxDateTimeInput provides support for localizing its input field and popup calendarfor different languages and date formats. Each localization is contained within its own file with the language code appended to the name, e.g. globalize.culture.de-DE.js for German. The desired localization file should be loaded along with the jqxdatetimeinput.js. The localization is set to the jqxDateTimeInput instance by setting the jqxDateTimeInput's culture property.
